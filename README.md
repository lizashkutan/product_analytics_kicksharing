# product_analytics_kicksharing
Данный проект выполнен в рамках тестового задания для стажировки по **Продуктовой аналитике** в Т-Банке.

## О проекте
Сервис "Самокаты" позволяет пользователям брать в аренду самокаты от партнера "Юрент" непосредственно через приложение Т-Банка.

## Ключевые этапы анализа
1.  **Разведочный анализ данных (EDA)**
    *   Оценка качества данных: проверка на пропуски, дубликаты, аномалии.
    *   Анализ основных метрик: количество поездок, пользователей, выручка.
    *   Исследование распределений: длительность и дистанция поездок.
    *   Анализ пользовательского поведения: сегментация, выявление паттернов.
    *   Визуализация результатов для наглядности.
2.  **Формулировка и проверка продуктовых гипотез**
    *   Было сформулировано *3 гипотезы*, направленные на улучшение ключевых метрик сервиса.
    *   Для каждой гипотезы проведена статистическая проверка.
    *   Выявлена и обоснована *наиболее перспективная гипотеза* для внедрения.
3.  **Анализ доли рынка**
    *   Проведена оценка доли сервиса Т-Банка в индустрии проката самокатов.
    *   Результаты представлены в контексте общего рынка.

## Технологии и инструменты
*   **Язык программирования:** Python 3
*   **Основные библиотеки:** Pandas, NumPy, Matplotlib, Seaborn
*   **Среда разработки:** Jupyter Notebook
*   **Визуализация и отчетность:** PowerPoint
